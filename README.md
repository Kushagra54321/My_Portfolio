## 🌟 Overview

This portfolio is designed to shatter the generic layout mold. It features a heavy focus on **cinematic aesthetics**, smooth Framer Motion animations, complex interactive layouts, and a neural network-inspired dark mode user interface. Every component, from the full-screen Hero intro to the interactive Project Modals, has been handcrafted for an immersive user experience.

## ✨ Features

- **Dark Mode Default:** Integrated with an immersive neural network/Earth nodes background.
- **Cinematic Hero Entry:** Full-screen intro with particle glowing effects, responsive text scaling, and profile image reveal.
- **Advanced Animations:** Leveraging `Framer Motion` for page scrolls, timeline stagger effects, and AnimatePresence modals.
- **11 Modular Sections:** Includes Hero, About, Experience Timeline, Interactive Projects Grid, Categorized Skills, Achievements, Flipping Certifications, Dashboard-style Coding Profiles, Education Gallery, and a fully functional Contact Form with heart animations.
- **Fully Responsive:** Completely optimized for mobile and tablet views without breaking the cinematic feel.
- **Vite-Powered:** Instant server start and lightning-fast HMR.

## 🛠️ Tech Stack

- **Frontend:** React 18, HTML5, Vanilla CSS 3
- **Animations:** Framer Motion, pure CSS keyframes
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Local Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Kushagra54321/My_portfolio.git
   cd My_portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
